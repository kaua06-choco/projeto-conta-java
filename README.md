O que faz: Criar conta, Depositar ,Sacar e Ver saldo
# 💰 Sistema de Conta Bancária

Projeto simples em Java para simular uma conta bancária.

## 🚀 Funcionalidades
- Criar conta
- Depositar dinheiro
- Sacar dinheiro
- Ver saldo

## 🧠 Tecnologias usadas
- Java

## ▶️ Como executar
1. Compile:
   javac Main.java

2. Execute:
   java Main

## 📌 Exemplo de saída
Saldo: 60.0
